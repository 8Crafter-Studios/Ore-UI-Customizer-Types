Types for 8Crafter's Ore UI Customizer.

This package is meant for use in plugin scripts, other Ore UI Customizer related modules, and internally within the Ore UI Customizer.

For scripts that would be run in-game, you should use the following modules:

-   [ore-ui-types](https://www.npmjs.com/package/ore-ui-types)
-   [@ore-ui-types/enums](https://www.npmjs.com/package/@ore-ui-types/enums)

For plugin scripts you should use the following modules in addition to this one:

-   [@ore-ui-customizer-api/plugin-env](https://www.npmjs.com/package/@ore-ui-customizer-api/plugin-env)
